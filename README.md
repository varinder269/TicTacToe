# 🎮 Tic Tac Toe (Java | OOP Design)

A clean and extensible implementation of the classic Tic Tac Toe game built using Java.
This project focuses on object-oriented design, modular architecture, and scalable code structure.

---

## 🚀 Features

* Configurable board size (not limited to 3x3)
* Clean separation of concerns (Board, Player, Game logic)
* Factory pattern for object creation
* Obersever for in app notifications
* Extensible design (easy to add AI or new rules)
* Input-driven gameplay

---

## 🧠 Design Highlights

* Object-Oriented Design with proper abstraction and encapsulation
* Factory Pattern used for creating players/symbols
* Modular structure for better scalability and maintainability

---

## 📁 Project Structure

src/main/java/
├── factory/
├── model/
├── service/
└── Main.java

---

## ▶️ How to Run

### Prerequisites

* Java 8 or above

### Steps

git clone https://github.com/varinder269/TicTacToe.git
cd TicTacToe

Compile:
javac -d out src/main/java/**/*.java

Run:
java -cp out Main

---

## 🎯 Gameplay

* Players take turns placing symbols on the board
* First player to complete a row, column, or diagonal wins
* Works for configurable board sizes
